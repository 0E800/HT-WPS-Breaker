## Here is how to make the script works

* Copy HT-WPS-Breaker.zip to Desktop .
* Open The Terminal .
* Type the following commands :

    - [X] cd Desktop
    - [X] unzip HT-WPS-Breaker.zip
    - [X] cd HT-WPS-Breaker
    - [X] chmod +x HT-WB.sh
    - [X] ./HT-WB.sh  or  bash HT-WB.sh
